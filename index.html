PrettyMining

Data Mining Library written in Scala

PrettyMining is a family of Scala libraries providing beautiful Scala APIs for data mining, include PrettyMatch for entity matching, PrettyBow for language modeling, text retrieval and classification and PrettyNeural for machine learning

All PrettyMining libraries are open source, and are available for Scala 2.10 and 2.11 under the GNU General Public License .

PrettyMatch

PrettyMatch is a String Matching library written in Scala. It contains many algorithms to calculate morphological, semantic and sounds-like distances. PrettyMatch can handle noisy data with good accuracy by using supervised Random Forest to combine and learn distances from all comparators.

PrettyBow

PrettyBow is a library of Scala for Statistical Language Modeling, Text Retrieval and Classification. It is a fast, feature abundant and highly configurable tookit built on the top of Lucene and Weka. The idea of PrettyBow is inspired by Bow (similar library written in C) and TagHelper (similar library written in Java)

PrettyNeural

PrettyNeural is a deep learning library written in Scala. include Multilayer Perceptron, Restricted Boltzmann Machine and Deep Belief Network. The algorithms can be applied on big data set by multithreading or distribution on Spark Framework . It can be a plugin for Weka as well.
