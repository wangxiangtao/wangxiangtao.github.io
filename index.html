PrettyMining

Intuitive libraries for everyday programming tasks in Scala

PrettyMining is a family of Scala libraries providing beautiful idiomatic Scala APIs for common programming tasks, like working with I/O, cryptography and JSON & XML processing.

All PrettyMining libraries are open source, and are available for Scala 2.10 and 2.11 under the Apache Software License 2.0. Source code is on GitHub, and binaries are available on Maven Central.

PrettyMatch

PrettyMatch is a String Matching library written in Scala. It contains many algorithms to calculate morphological, semantic and sounds-like distances for measurement of string similarity. Using an supervised Random Forest algorithm to combine and learn features from all comparators for matching, PrettyMatch can handle noisy data with good accuracy.

PrettyBow

PrettyBow is a library of Scala for Statistical Language Modeling, Text Retrieval and Classification. It is a fast, feature abundant and highly configurable tookit built on the top of Lucene and Weka. The idea of PrettyBow is inspired by Bow (similar library written in C) and TagHelper (similar library written in Java)

Typesafe

The power to implement and refactor code confidently comes from the expressiveness of Scala’s type system. PrettyMining uses extensive typing to help exploit the guidance and constraints Scala’s type system provides.
